# Circle Effect

## Settings

| Key              | Expects | Description                               |
| ---------------- | ------- | ----------------------------------------- |
| radius           | double  | The range at which particles may generate |
| xRotation        | double  |                                           |
| yRotation        | double  |                                           |
| zRotation        | double  |                                           |
| angularVelocityX | double  |                                           |
| angularVelocityY | double  |                                           |
| angularVelocityZ | double  |                                           |
| enableRotation   | boolean |                                           |

## Example

```YML
        1:
          class: Circle
          category: PERSISTENT
          radius: 0.8
```
