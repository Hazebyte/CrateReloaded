# Original Effect

## Settings

| Key    | Expects | Description                               |
| ------ | ------- | ----------------------------------------- |
| radius | integer | The range at which particles may generate |

## Example

```YML
        1:
          class: Original
          category: PERSISTENT
          particle: VILLAGER_HAPPY
          radius: 0.8
          particleCount: 10
          period: 1
```
